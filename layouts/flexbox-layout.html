O que é Flexbox?

CSS Flexible Box Layout (Flexbox) é um modelo de layout unidimensional projetado para distribuir espaço entre itens e alinhar-os de formas complexas. Diferente do layout tradicional baseado em blocos (que trabalha em cima e baixo), o Flexbox trabalha em eixos.

📊 Quando usar Flexbox:
  • Alinhar itens horizontal ou verticalmente no centro
  • Criar barras de navegação
  • Distribuir cards igualmente em uma linha
  • Criar sidebars responsivos
  • Layouts de formulários alinhados
⚠️ Limitação: Flexbox é unidimensional (uma direção por vez: linha OU coluna). Para layouts bidimensionais (linha E coluna simultaneamente), use CSS Grid.

Conceitos Fundamentais

Flex Container
display: flex
EIXO
flex-direction

Define a direção do eixo principal: horizontal (row) ou vertical (column).

PRINCIPAL
justify-content

Alinha itens ao longo do eixo principal (direção do flex).

TRANSVERSAL
align-items

Alinha itens ao longo do eixo transversal (perpendicular ao flex).

ESPAÇAMENTO
flex-grow/shrink

Controla como os itens crescem ou encolhem para ocupar espaço disponível.

Propriedades do Container

Ativando o Flexbox

Display Flex 
.container {
  display: flex;        /* Bloco flexível */
  /* ou */
  display: inline-flex; /* Inline flexível */
}
💡 Dica: display: flex cria um block-level flex container (ocupa largura total). inline-flex cria um container inline que se ajusta ao conteúdo.

align-self: Alinhamento Individual

Sobrescrevendo o alinhamento de um item específico 
.container {
  display: flex;
  align-items: center;  /* Todos centralizados */
}

.item-destaque {
  align-self: flex-end; /* Exceto este, que vai para baixo */
}

Propriedades dos Itens

flex-grow: Fator de Crescimento

Define quanto um item pode crescer em relação aos outros para preencher espaço vazio.

Exemplo flex-grow 
.container {
  display: flex;
}

.item1 { flex-grow: 1; }  /* Ocupa 1 parte */
.item2 { flex-grow: 2; }  /* Ocupa 2 partes (dobro do item1) */
.item3 { flex-grow: 1; }  /* Ocupa 1 parte */

/* Resultado: item2 é 2x maior que item1 e item3 */

flex-shrink: Fator de Encolhimento

Define quanto um item pode encolher quando não há espaço suficiente. Padrão é 1 (pode encolher). 0 = não encolhe.

flex-basis: Tamanho Inicial

Define o tamanho inicial do item antes da distribuição do espaço. Pode ser em px, %, rem, ou auto.

⚠️ flex-basis vs width: Em flex-direction: row, flex-basis controla a largura. Em column, controla a altura. Use flex-basis sempre que possível no contexto flex!

Shorthand: flex

Sintaxe completa 
.item {
  /* flex: grow shrink basis */
  flex: 0 1 auto;    /* Padrão: não cresce, pode encolher, tamanho auto */
  flex: 1;           /* Cresce igual, encolhe igual, basis 0% */
  flex: 0 0 200px;   /* Tamanho fixo de 200px, não cresce nem encolhe */
  flex: 1 1 300px;   /* Começa com 300px, pode crescer e encolher */
}

Valores Comuns de flex

flex: 1 Distribui espaço igualmente entre todos os itens
flex: 0 0 auto Tamanho baseado no conteúdo, não cresce nem encolhe
flex: 0 0 200px Tamanho fixo de 200px (útil para sidebars)
flex: 1 0 0 Mesmo tamanho para todos, ignorando conteúdo

order: Reordenando Visualmente

Controlando a ordem visual 
.item1 { order: 2; }  /* Aparece em segundo */
.item2 { order: 3; }  /* Aparece em terceiro */
.item3 { order: 1; }  /* Aparece em primeiro */

/* Padrão é 0. Valores negativos são permitidos */
♿ Acessibilidade: order altera apenas a ordem visual, não a ordem do DOM ou da navegação por teclado. Mantenha uma ordem lógica no HTML e use order apenas para ajustes visuais menores.

Quebra de Linha (flex-wrap)

Por padrão, flex items tentam caber em uma única linha. flex-wrap permite que quebrem para múltiplas linhas.

align-content: Alinhamento das Linhas

Quando há múltiplas linhas (wrap), controla o espaçamento entre elas no eixo transversal.

Container com wrap 
.container {
  display: flex;
  flex-wrap: wrap;           /* Permite quebra de linha */
  align-content: center;     /* Alinha linhas no centro vertical */
  gap: 20px;                 /* Espaço entre itens e linhas */
}

Propriedade Gap

A forma moderna de adicionar espaçamento entre flex items, sem necessidade de margens complexas ou hacks.

Sintaxe Gap 
.container {
  /* Gap igual em todas as direções */
  gap: 20px;
  
  /* Gap diferente para row e column */
  gap: 10px 30px;  /* row-gap: 10px; column-gap: 30px; */
  
  /* Ou específico */
  row-gap: 20px;
  column-gap: 40px;
}

Holy Grail Layout com Flexbox

Layout Clássico: Header, Footer, 3 Colunas 
<style>
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
  margin: 0;
}

main {
  display: flex;
  flex: 1;  /* Ocupa todo espaço disponível */
}

header, footer {
  background: #333;
  color: white;
  padding: 20px;
}

nav, aside {
  flex: 0 0 200px;  /* Largura fixa */
  background: #f0f0f0;
  padding: 20px;
}

article {
  flex: 1;  /* Ocupa espaço restante */
  padding: 20px;
}

/* Responsivo: empilha em telas pequenas */
@media (max-width: 768px) {
  main {
    flex-direction: column;
  }
  nav, aside {
    flex: 0 0 auto;
  }
}
</style>

<body>
  <header>Cabeçalho</header>
  <main>
    <nav>Menu</nav>
    <article>Conteúdo Principal</article>
    <aside>Publicidade</aside>
  </main>
  <footer>Rodapé</footer>
</body>
