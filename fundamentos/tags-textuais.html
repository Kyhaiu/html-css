Introdução

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. As tags textuais são elementos fundamentais que dão estrutura e significado ao conteúdo escrito, permitindo que navegadores interpretem e exibam informações de forma organizada e acessível.

Por que aprender tags textuais?

  • 📚 Estruturação: Organize conteúdo de forma hierárquica e lógica
  • ♿ Acessibilidade: Torne seu site utilizável por leitores de tela
  • 🔍 SEO: Melhore o posicionamento nos mecanismos de busca
  • 📱 Responsividade: Garanta compatibilidade com diferentes dispositivos
  • 🎨 Estilização: Base para aplicação de CSS e JavaScript
💡 Dica Pro: Sempre prefira tags semânticas (como <article>, <section>) em vez de genéricas (<div>, <span>) quando possível. Isso melhora a acessibilidade e o SEO do seu site.

Títulos e Cabeçalhos

Os elementos de cabeçalho (<h1> a <h6>) representam seis níveis de títulos, sendo <h1> o mais importante e <h6> o menos importante. Eles são cruciais para a estrutura do documento e SEO.

⚠️ Importante: Use apenas um <h1> por página e mantenha a hierarquia sequencial (não pule de h2 para h4, por exemplo).
HTML - Estrutura de Títulos 
<h1>Título Principal da Página</h1>
<h2>Seção Importante</h2>
<h3>Subseção Específica</h3>
<h4>Tópico Detalhado</h4>
<h5>Subtópico</h5>
<h6>Nota de Rodapé</h6>
Resultado Visual

Seção Importante (h2)

Subseção Específica (h3)

Tópico Detalhado (h4)

Subtópico (h5)
Nota de Rodapé (h6)

Atributos Globais Relevantes

  • id - Identificador único para navegação e CSS/JS
  • class - Classes para estilização em grupo
  • style - Estilos inline (evite usar, prefira CSS externo)

Formatação de Texto

Tags de formatação podem ser divididas em visuais (apenas aparência) e semânticas (significado + aparência). Sempre prefira as semânticas!

Tag Visual (Evitar) Tag Semântica (Preferir) Uso Correto
<b> - Negrito <strong> - Importância Alertas, avisos críticos
<i> - Itálico <em> - Ênfase Destaque em frases
<u> - Sublinhado <ins> - Inserção Texto adicionado/editado
<s> - Tachado <del> - Remoção Texto removido/obsoleto

Exemplos de Uso

Formatação Semântica 
<!-- Ênfase e Importância -->
<p>Você <em>deve</em> ler isto. É <strong>urgente</strong>!</p>

<!-- Marcações de Edição -->
<p>Preço: <del>R$ 100,00</del> <ins>R$ 79,90</ins></p>

<!-- Sobrescrito e Subscrito -->
<p>Fórmula: H<sub>2</sub>O | Matemática: E=mc<sup>2</sup></p>

<!-- Destaque e Código -->
<p>Use o comando <kbd>Ctrl</kbd> + <kbd>S</kbd> para salvar.</p>

<!-- Pequeno e Marcação -->
<p><small>Copyright 2024</small> | 
<mark>Texto destacado</mark></p>
Resultado Visual

Você deve ler isto. É urgente!

Preço: R$ 100,00 R$ 79,90

Fórmula: H2O | Matemática: E=mc2

Use o comando Ctrl + S para salvar.

✅ Boas Práticas: Use <mark> para destacar resultados de busca, <kbd> para atalhos de teclado, e <samp> para saídas de programas.

Tags Semânticas de Texto

HTML5 introduziu tags que dão significado ao conteúdo, melhorando acessibilidade e SEO:

<abbr>

Abreviações e acrônimos. Use o atributo title para o significado completo.

HTML é uma linguagem de marcação.
<address>

Informações de contato/autoria. Geralmente renderiza em itálico.

Escrito por João Silva.
Visite-nos em: exemplo.com
<time>

Representa datas/horários. Atributo datetime para máquinas.

Evento em 25 de Dezembro.
<dfn>

Definição de termo. Indica a primeira definição de um conceito.

HTML é a linguagem padrão da web.
Exemplo Completo de Semântica 
<article>
  <header>
    <h1>Novo Padrão Web</h1>
    <p>Publicado em <time datetime="2024-01-15">15 de Janeiro</time></p>
  </header>
  
  <p>A <dfn><abbr title="Cascading Style Sheets">CSS</abbr></dfn> 
  é usada para estilizar documentos.</p>
  
  <address>
    Autor: Maria Developer<br>
    Email: maria@exemplo.com
  </address>
</article>

Listas

Listas organizam itens relacionados. Existem três tipos principais:

<ul> + <li>

Lista não ordenada (bullets). Use para itens sem sequência específica.

  • Item A
  • Item B
  • Item C
<ol> + <li>

Lista ordenada (numerada). Use para procedimentos ou rankings.

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo
<dl> + <dt> + <dd>

Lista de definições. Termo (dt) e descrição (dd).

HTML
Linguagem de marcação
CSS
Linguagem de estilos

Atributos Especiais de <ol>

Lista Ordenada Avançada 
<!-- type: 1, A, a, I, i -->
<ol type="I" start="5">
  <li>Item V</li>
  <li value="10">Item X (forçado)</li>
  <li>Item XI (continua)</li>
</ol>

<!-- Lista reversa -->
<ol reversed>
  <li>Top 3</li>
  <li>Top 2</li>
  <li>Top 1</li>
</ol>
Resultado: Lista Reversa
  1. Top 3
  2. Top 2
  3. Top 1

Links e Âncoras

A tag <a> (âncora) cria hyperlinks, elemento fundamental da web.

Tipos de Links 
<!-- Link externo -->
<a href="https://developer.mozilla.org" target="_blank" rel="noopener">
  MDN Web Docs
</a>

<!-- Link interno/âncora -->
<a href="#secao-contato">Ir para Contato</a>

<!-- Link de email -->
<a href="mailto:contato@exemplo.com?subject=Dúvida">
  Enviar email
</a>

<!-- Link de telefone -->
<a href="tel:+5511999999999">Ligar agora</a>

<!-- Download -->
<a href="arquivo.pdf" download="guia.pdf">
  Baixar PDF
</a>

Atributos Importantes

  • href - URL de destino (obrigatório)
  • target="_blank" - Abre em nova aba (sempre use com rel="noopener")
  • rel - Relacionamento (nofollow, noopener, noreferrer)
  • download - Força download em vez de navegação
  • title - Tooltip ao passar o mouse
🔒 Segurança: Sempre adicione rel="noopener noreferrer" em links target="_blank" para evitar vulnerabilidades de tabnabbing.

Citações

<blockquote>

Citação em bloco (longa). Geralmente com recuo e itálico.

"A web é mais uma criação social do que técnica." — Tim Berners-Lee
<q>

Citação inline (curta). Adiciona aspas automaticamente.

Ele disse HTML é fascinante durante a aula.
<cite>

Referência à obra citada (título, autor, URL).

O Senhor dos Anéis por J.R.R. Tolkien.
Exemplo de Citação Completa 
<figure>
  <blockquote cite="https://developer.mozilla.org">
    <p>O HTML (HyperText Markup Language) é o bloco de construção 
    mais básico da web.</p>
  </blockquote>
  <figcaption>
    — MDN Web Docs, <cite>HTML: Linguagem de Marcação de Hipertexto</cite>
  </figcaption>
</figure>

Fragmento de código inline. Fonte monoespaçada.

Use a função console.log() para debugar.
<pre>

Texto pré-formatado. Preserva espaços e quebras de linha. 

Linha 1
    Linha 2 (indentada)
Linha 3
<samp>

Saída de programa ou computador.

Erro 404: Página não encontrada
<var>

Variáveis em contexto matemático ou de programação.

A área é calculada por π × r².
