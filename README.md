> Repositório educacional com exemplos práticos, exercícios e projetos para aprendizado de HTML5 puro.

---

## 🎯 Objetivo

Este repositório serve como material de apoio para estudantes de desenvolvimento web, focando exclusivamente em **HTML5 semântico** e **boas práticas**, sem dependências de frameworks ou bibliotecas externas.

### O que você vai aprender:
- ✅ Estruturação semântica de documentos HTML
- ✅ Estruturação de layouts responsivos
- ✅ Tags textuais e de formatação
- ✅ Formulários acessíveis e validados
- ✅ Tabelas de dados responsivas
- ✅ Multimídia (imagens, áudio, vídeo)
- ✅ Acessibilidade (A11y) e SEO básico
- ✅ Padrões modernos de desenvolvimento

---

## 🚀 Começando

### Pré-requisitos
- Navegador moderno (Chrome, Firefox, Edge, Safari, Outros)
- Editor de código (VS Code *recomendado*, Sublime, Vim, Neo Vim, etc.)
- Git (opcional, para clonar)

### Instalação

```bash
# Clone o repositório
git clone https://github.com/Kyhaiu/html-css.git

# Entre na pasta
cd html-css

# Abra os arquivos .html no navegador
# Ou use uma extensão como Live Server no VS Code
```
